# git 技术手册

Git是一个开源的分布式版本控制系统，可以有效、高速地处理从很小到非常大的项目版本管理。 git是Linus（Linux之父）为了帮助管理Linux内核开发而开发的一个开放源码的版本控制软件，目前被各大企业广泛应用于代码托管和协作开发中。

本手册收录了一些git常用场景及指令教程。
